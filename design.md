## Typography Hierarchy

```
┌─────────────────────────────────────────────┐
│ H1 (Name): 3rem (48px) - Bold              │
│ H2 (Section Titles): 2.5rem (40px) - Bold  │
│ H3 (Project Titles): 1.5rem (24px) - Bold  │
│ Body Text: 1rem (16px) - Regular           │
│ Small Text: 0.875rem (14px) - Regular      │
│                                             │
│ Font Stack: 'Inter', sans-serif            │
│ Fallback: system-ui, -apple-system         │
└─────────────────────────────────────────────┘
```

## Key Design Features

- **Single page layout** with smooth scrolling navigation
- **Card-based project display** for easy scanning
- **Responsive grid system** (3 columns → 1 column on mobile)
- **Clean form design** with proper spacing and validation states
- **Consistent spacing** using 8px grid system
- **Hover effects** on interactive elements
- **Mobile-first approach** with touch-friendly buttons
- **Professional imagery** with consistent aspect ratios
